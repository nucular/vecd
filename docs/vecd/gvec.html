• vecd

    Struct gvec

    Provides a generic template for vector-like data types.

    Makes use of SIMD types if available and falls back to a static-sized array if not.

    Constructors

    Name Description
    this Initialize the components of the vector to multiple scalars using varargs.
    this Initialize the vector to the contents of another vector of the same type and size.
    this Initialize the vector to an array of the same type as the vector.
    this Initialize all components of the vector to a scalar.

    Methods

    Name Description
    cross Returns the cross product of two 3D vectors.
    dot Returns the dot product of two vectors.
    dup Returns a duplicate of the vector.
    length Returns the length/magnitude of the vector.
    length2 Returns the squared length/magnitude of the vector.
    normalize Normalizes the vector in-place.
    normalized Returns a normalized copy of the vector.
    opApply Implements the apply operator for foreach iteration with one parameter.
    opApply Implements the apply operator for foreach iteration with two parameters.
    opApplyReverse Implements the apply operator for foreach_reverse iteration with one parameter.
    opApplyReverse Implements the apply operator for foreach_reverse iteration with two parameters.
    opAssign Implements the assignment operator with a scalar of the type of the vector or a type that can be casted to it implicitely.
    opAssign Implements the assignment operator with an array of the type of the vector (or a type that can be casted to it implicitely).
    opAssign Implements the assignment operator with an array of the type of the vector (or a type that can be casted to it implicitely) and the same number of components.
    opDollar Implements the dollar operator.
    opIndex Implements the index operator.
    opIndexAssign Implements the index assignment operator.

    Aliases

    Templates

    Name Description
    opBinary Implements all binary operators on two vectors of the same length and type.
    opBinary Implements all binary operators with a scalar of the type of the vector or a type that can be casted to it implicitely.
    opIndexOpAssign Implements all index op assignment operators.
    opOpAssign Implements all op assignment operators on two vectors of the same length and type.
    opOpAssign Implements all op assignment operators with a scalar of the type of the vector or a type that can be casted to it implicitely.
    opUnary Implements all unary operators on a vector.
